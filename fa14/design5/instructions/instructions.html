CSS Positioning & Floats
Required Tools: Google Chrome, Internet, text editor, a cute picture (if you don't want to use ours).
For this hands-on, you will be making a Medium like page for a blog post to practice positioning.
Assignment

You want your blog post to have a giant image like on Medium and also some text for where you voiced your inner most feelings. On top, there are some words that are supposed to be located at the top bar and a few lines of text, the titles, that are supposed to overlay ontop the big image. But, right now it looks like this. Let's style it so it looks way better!

1. Style the Top

If you look at the page now, the cat picture is gigantic and the navgiation bar needs work. To make sure the cat image is stretched across the top of the page but not too big, let's set .blog-image width to 100%.

Great! The cat looks nice (and cute), but to make navigation easier for the user, let's make it a side bar that sticks to the side of the page. Give navbar position: fixed to fix it to the left. Now your "sidebar" is a small rectangle on the top. To make it stretch down the side, make the width 200px and height 100% under navbar too. Then to make the text go down the sidebar instead of across change display to block.


2. Style the Titles

Underneath the navigation bar, notice there are some text there. That is actually the title and subtitle. However, we don't want it underneath the bar, we want it ontop our image so we can actually see it. In the CSS class .blog-title we can margins to move our text around and using font-weight, color, and size to modify the text appearance:

    font-weight: 700;
    font-size: 50px;
    color: white;
    position: absolute;
    margin-top: 300px;
    margin-left: 300px;

Let's do the same thing to the subtitle as well under .blog-timestamp

    font-style: italic;
    position: absolute;
    color: white;
    margin-top: 370px;
    margin-left: 300px;


Step 3: Style the Body Text

Great! Now, let's make that ugly chunk of words on the bottom a bit nicer. First, let's make the text actually readable. To do this, apply line height and a margin to separate it from the image on top to .blog-lorem in the css.

    line-height: 1.5em;
    font-weight: 300;
    margin: 10px 0;

Now even though the text is legible, it stretches across the entire page which we don't want. If you are familiar with Medium, the site keeps the main blog content in the center of the page with ample space on the sides. All you have to do is to apply a margin and width to .blog-text:

    width: 720px;
    margin: 40px auto;



Bonus: Style the Button

Great! Your blog is beautiful now. But, if you noticed at the very bottom, there is a "Share!" text. That is what is supposed to be a button for you to share your lovely post with your friends. Using your knowledge from the previous weeks, apply some style to it to make it pretty. This is what we did:

button {
    background: black;
    color: white;
    width: 100px;
    text-align: center;
    padding: 7px;
    border-radius: 2px;
    cursor: pointer;
} .button-share {
    background: #6392c0;
    border: 1px solid black;
    -webkit-transition: 0.5s;
    margin-top: 20px;
} .button-share:hover {
    background: rgba(99, 146, 192, 0.7);
}
