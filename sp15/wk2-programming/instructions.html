Assignment

Time to put the HTML you learned to use! For this hands-on, we will be making a simple registration page for a dating site.

At the end of this hands-on, you should have something like the page above.

Step 1: Set Up HTML

Every HTML file should have the following basic structure:
<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>

Step 2: Add Content

Start making your registration page by giving it a heading. Then, using <input> and <select> tags, provide the user with space to type their first name, and choose their gender, their gender preference, and their country using a dropdown menu. Put these options in a cohesive sentence like in the example page above. Lastly, make a "Submit" button using the <button> tag.

Step 3: Image and Style

Add an image before your text to spice this page up.
If you refresh your page, you'll likely see that your image is HUGE. Fix this by adding some CSS.
Create a new file and call it "style.css". Link to this file in your html using:

<link rel="stylesheet" type="text/css" href="(path to style.css file)">

Put this inside the <head> tag of your HTML file.
Then, select the image and change its height to be around 300px.
